# AI Resume Parser (NLP Project)

This project is an AI-based Resume Parser that extracts useful information from resumes using Natural Language Processing (NLP). The system processes PDF resumes and identifies key information such as skills, contact details, and candidate information.

## Features

- Resume parsing using NLP
- Extract candidate details
- Extract skills from resume
- Extract email and phone number
- Streamlit web interface

## Technologies Used

- Python
- spaCy NLP
- PyResParser
- Streamlit
- Natural Language Processing

## Project Workflow

1. Upload resume file (PDF)
2. The NLP model processes the resume text
3. Important information such as skills and contact details are extracted
4. Results are displayed in the interface

## Installation

Clone the repository:

git clone https://github.com/karthiktunuguntla/AI-NLP-RESUME-PARSER.git

Install required libraries:

pip install -r requirements.txt

Run the application:

streamlit run main.py
